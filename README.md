# easy-open-router-chat

## How to use

Download the code [or clone the repo]

Install it and run:

```bash
pnpm install
pnpm start
```

## Setup env

### Copy .env.example to .env

```bash
cp .env.example .env
```

### Fill in the .env file

Set environment variables with your own values
